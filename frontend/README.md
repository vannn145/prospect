# Front-end do Prospect

Aplicação React + TailwindCSS para prospecção de empresas.

## Executar em desenvolvimento

```powershell
npm install
npm run dev
```

Aplicação padrão: `http://localhost:5173`

## Variáveis de ambiente

Crie o arquivo `.env` com:

```env
VITE_API_URL=http://localhost:4000/api
```

## Build de produção

```powershell
npm run build
npm run preview
```
